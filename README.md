# 项目介绍
“随你记” 微信小程序，主要是为了解决学生能够更好的了解自己的收支。采用小程序的形式，无需下载app，随记随走，非常便捷。后端采用云开发，前端用了Echarts框架，原生框架等。在2019年的微信小程序高校大赛中，获得华南赛区二等奖。
